# Better Monitor Agent

## 项目简介
本项目是 [Better Monitor](https://github.com/EnderKC/BetterMonitor) 的 Android 端 agent 第三方实现

## 目前已支持功能
- [x] 系统信息
- [x] 内存使用率
- [x] 磁盘使用率
- [x] 落地 IP 地址
- [x] 文件管理
- [x] 终端模拟器
- [x] Root / 非 Root 兼容
- [x] 后台运行

## 后期规划
未来可能会增加更多 Android 相关特色功能，比如电量等信息上报

## 致谢
本项目基于 [Better Monitor](https://github.com/EnderKC/BetterMonitor) 的 WebSocket 协议开发，感谢作者的付出
